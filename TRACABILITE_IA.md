# 🤖 Traçabilité de l'usage de l'IA

> **Règle de la fiche :** Tout usage de l'IA doit être tracé, vérifié et assumé par l'équipe.
> Chaque entrée doit contenir : l'objectif, ce que l'IA a produit, ce que l'équipe a corrigé/validé, et la preuve (référence de commit ou extrait).

---

## Comment remplir ce fichier

| Colonne | Ce qu'on y met |
|---------|---------------|
| **Usage** | Génération de code / Conception / Tests / Documentation |
| **Outil IA** | Claude / ChatGPT / GitHub Copilot / autre |
| **Objectif** | Ce qu'on lui a demandé |
| **Ce que l'IA a produit** | Résumé de l'output |
| **Corrections humaines** | Ce qu'on a modifié, refusé ou vérifié |
| **Preuve** | Référence de commit, nom de fichier, extrait de code |
| **Membre** | Qui a utilisé l'IA |
